### 二、公用运算符
python3中有一些运算符的公用的，如下表所示：  

| 运算符 |      描述      |      支持的容器类型      |  
| :----: | :------------: | :----------------------: |  
|   +    |      合并      |    字符串、列表、元组    |  
|   *    |      复制      |    字符串、列表、元组    |  
|   in   |  元素是否存在  | 字符串、列表、元组、字典 |  
| not in | 元素是否不存在 | 字符串、列表、元组、字典 |  

#### 1、+ 号运算符的用法
``` python
# 1. 字符串
str1 = 'aa'
str2 = 'bb'
str3 = str1 + str2
print(str3)  # 输出结果为：aabb

# 2. 列表
list1 = [1, 2]
list2 = [10, 20]
list3 = list1 + list2
print(list3)  # 输出结果为：[1, 2, 10, 20]

# 3. 元组
t1 = (1, 2)
t2 = (10, 20)
t3 = t1 + t2
print(t3)  # 输出结果为：(10, 20, 100, 200)
```
#### 2、* 号运算符的用法
``` python
# 1、字符串
str = '还钱 '
print(str*4)

# 2、列表
list = ["hello "]
print(list*4)

# 3、元组
t1 = ("word ")
print()
```
### 3、in或not in 的用法
``` python
# 1. 字符串 验证a是否存在
str = "abcd"
print('a' in str)  # True
print('a' not in str)  # False

# 2. 列表 验证a是否存在
list = ['a', 'b', 'c', 'd']
print('a' in list)  # True
print('a' not in list)  # False

# 3. 元组 验证a是否存在
t1 = ('a', 'b', 'c', 'd')
print('a' in t1)  # True
print('a' not in t1)  # False

# 4、字典 验证name是否存在
dict = {"name": "不觉","age": 18}
print("name" in dict)
print("name" not in dict)
```
### 三、公用的方法
python3中有一些方法也是公用的，如下表所示：  

| 函数                    | 描述                                                         |  
| ----------------------- | ------------------------------------------------------------ |  
| len()                   | 计算容器中元素个数                                           |  
| del 或 del()            | 删除                                                         |  
| max()                   | 返回容器中元素最大值                                         |  
| min()                   | 返回容器中元素最小值                                         |  
| range(start, end, step) | 生成从start到end的数字，步长为 step，供for循环使用           |  
| enumerate()             | 函数用于将一个可遍历的数据对象(如列表、元组或字符串)组合为一个索引序列，同时列出数据和数据下标，一般用在 for 循环当中。 |  

#### 1、len()函数
实例：  
``` python
# 1、字符串
str = "abcd"
print(len(str)) # 4

# 2、列表
list = ["a","b","c","d"]
print(len(list)) # 4

# 3、元组
t1 = ("a","b","c","d")
print(len(t1)) # 4

# 4、字典
dict = {"name": "小红","age": 18}
print(len(dict)) # 2

# 5、集合
s1 = {10,20,30}
print(len(s1)) # 3
```
#### 2、del()函数
实例：  
``` python
# 1、字符串
str1 = "abcd"
del str1
print(str1) # 报错not defined --因为str1已被删除

# 2、列表
list1 = ["aa","bb","cc","dd"]
# del(list)  # 删除整个列表 -- 输出为报错not defined
del(list1[0]) # 删除索引为0的数据
print(list1) # ['bb', 'cc', 'dd']

# 3、元组
t1 = ("a","b","c","d")
del(t1) # 删除t1整个元组
print(t1) # not defined

# 4、字典
dict = {"name": "岳绮罗","age": 18}
del(dict["name"]) # 删除键对值name
print(dict) # 输出结果为{'age': 18}
```
### 3、max()函数和min()函数
实例：  
``` python
# 1、字符串
str1 = "abkgyv"
# 最大值
print(max(str1)) # 输出结果为y
# 最小值
print(min(str1)) # 输出就结果为a

# 2、列表
list1 = [1,2,3,4,5,6]
# 最大值
print(max(list1)) # 输出结果为6
# 最小值
print(min(list1)) # 输出结果为1

# 3、元组
t1 = (1,2,3,4,8,9)
# 最大值
print(max(t1)) # 输出结果为9
# 最小值
print(min(t1)) # 输出结果为1

# 4、字典
dict1 = {"k1": 1,"k2": 2,"k3": 3,"k4": 4}
# 最大值
print(max(dict1)) # 输出结果为k4
# 最小值
print(min(dict1)) # 输出结果为k1

# 5、集合
s1 = {5,6,20,55,44,66}
# 最大值
print(max(s1)) # 输出结果为66
# 最小值
print(min(s1)) # 输出结果为5
```
#### 4、range()  
描述：生成从start到end的数字，步长为 step，供for循环使用  

语法：  
``` python
range(开始数字,结束数字,步长)
```
实例：  
``` python
# 步长为为1
for i in range(1, 10, 1):
    print(i) # 输出结果123456789

# 步长为2
for i in range(1, 10, 2):
    print(i) # 输出结果：1 3 5 7 9

# 不写开始数字和步长
for i in range(10):
    print(i) # 输出结果任然是0123456789

# 1、如果不写开始默认冲0开始
# 2、如果不写步长，则默认为1
# 3、生成的数字不包涵结束数字
```
#### 5、enumerate()
描述：enumerate() 函数用于将一个可遍历的数据对象(如列表、元组或字符串)组合为一个索引序列，同时列出数据和数据下标，一般用在 for 循环当中    
语法：  
``` python
enumerate(可遍历对象, start=0) 
```
注意：start参数用来设置遍历数据的下标的起始值，默认为0。（一般都把start省略不写）  
实例：  
``` python
# enumerate输出结果是元组，元组的第一个数据是原迭代对象对应的下标，第二个则是原迭代对象的数据
list1 = ['a','n','c','d','e']

for i in enumerate(list1):
    print(i)
for a in enumerate(list1):
    print(a)

# 输出结果
# 第一个for循环结果       # 第二个for循环结果
        (0, 'a')        (1, 'a')  
        (1, 'n')        (2, 'a')  
        (2, 'c')        (3, 'a')  
        (3, 'd')        (4, 'a')  
        (4, 'e')        (5, 'a')  
```
### 四、数据类型转换
为什么要做数据类型的转换？  
因为有时候程序里已有的程序类型和程序想要的类型不一致。  
#### 1、tuple()

作用：将某个序列转换成元组  
实例：  
``` python
list1 = [10,20,30,40,50]
s1 = {100, 200, 300, 400, 500}

print(tuple(list1))  # 输出结果：(10, 20, 30, 40, 50)
print(tuple(s1)) # 输出结果：(100, 200, 300, 400, 500)
```
#### 2、list()

作用：将某个序列转换成列表  
实例：  
``` python
tuple1 = (10,20,30,40,50)
s1 = {100, 200, 300, 400, 500}

print(list(tuple1))  # 输出结果：[10, 20, 30, 40, 50]
print(list(s1)) # 输出结果：[100, 200, 300, 400, 500]
```
#### 3、set()

描述：将某个序列转换成列表
实例：  
``` python
tuple1 = (10,20,30,40,50)
list1 = [100, 200, 300, 400, 500]

print(set(tuple1))  # 输出结果：{40, 10, 50, 20, 30}
print(set(list1)) # 输出结果：{100, 200, 300, 400, 500}

# 1、 集合可以快速完成列表去重
# 2.、集合不支持下标
```
